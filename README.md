### The task
```
Naudojant bet kokius įrankius sukurti mažą klientų valdymo frontend aplikaciją.

Pagrindiniai punktai:
1. Esamų klientų sąrašas;
2. Naujo kliento forma;
3. Galimybė pakoreguoti esamus duomenis;
4. Galimybė ištrinti esamus duomenis;
5. Duomenis saugoti naudojant naršyklės atmintinę;
6. Kliento formos duomenys:
  6.1. Vardas, pavardė;
  6.2. Telefono numeris;
  6.3. Adresas (šalis, miestas, gatvė, namo numeris, pašto kodas).
  6.4. Adreso prefillui panaudoti Google Places API ar bet kokią kitą alternatyvą.

Naudingi, bet neprivalomi punktai:
 1. Stilizavimas naudojant css ar bet kokį css preprocesorių.
```
